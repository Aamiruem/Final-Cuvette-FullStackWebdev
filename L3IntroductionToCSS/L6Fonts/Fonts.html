Font Properties

CSS provides several properties to control the appearance of text. Here are the most important ones:

font-family

Specifies the font for an element. It's a prioritized list of font family names.

font-family: "Gill Sans", sans-serif;
This text uses Gill Sans font
This text uses Times New Roman font

font-size

Sets the size of the font. Can be specified in pixels, ems, rems, percentages, or other units.

font-size: 24px;
Size 16px
Size 24px
Size 32px

font-weight

Sets the weight (or boldness) of the font. Values range from 100 to 900, or keywords like normal and bold.

font-weight: bold;
Weight 300 (Light)
Weight 400 (Normal)
Weight 700 (Bold)

font-style

Specifies the font style for text. Most commonly used for italic text.

font-style: italic;
Normal style
Italic style
Oblique style

line-height

Sets the amount of space used for lines, such as in text. It's often used to set the vertical rhythm of text.

line-height: 1.6;

Font Loading Best Practices

To ensure your fonts load quickly and don't cause layout shifts, follow these best practices:

/* Preload critical fonts */ @font-face { font-family: 'MyFont'; src: url('myfont.woff2') format('woff2'); font-display: swap; /* Optional: controls how fonts are displayed */ } /* Use font-display: swap for better performance */ body { font-family: 'MyFont', sans-serif; }
  • Use WOFF2 format for better compression
  • Preload critical fonts
  • Use font-display: swap to avoid invisible text
  • Limit the number of font weights and styles you use
  • Consider using system fonts for better performance
